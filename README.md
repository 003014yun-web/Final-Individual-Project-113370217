## Selected Research Question

**Research Question:**

Do HIV education, condom use, and HIV testing predict the number of sexual partners among sexually active American adolescents?

---

## Variables

### Response Variable

**MultipleSexPartners**

- Number of sexual partners among sexually active adolescents.
- Recoded into a numerical scale from 1 to 5.

### Predictor Variables

**TaughtAboutHIV**
- Received HIV education in school.
- Yes = 1, No = 0.

**HIVTesting**
- Ever tested for HIV.
- Yes = 1, No = 0.

**CondomUse**
- Used a condom during the last sexual intercourse.
- Yes = 1, No = 0.

---

## Statistical Method

This study used **Multiple Linear Regression** to examine whether HIV education, HIV testing, and condom use significantly predict the number of sexual partners among sexually active adolescents.

Regression model:

MultipleSexPartners = β₀ + β₁(TaughtAboutHIV)
                     + β₂(HIVTesting)
                     + β₃(CondomUse)
                     + ε

The model was evaluated using:

- Regression coefficients (β)
- t-tests
- F-test
- R²
- p-values
- 95% confidence intervals

---

## Short Final Conclusion

The regression model was statistically significant.

Main findings:

- HIV education was associated with fewer sex partners.
- HIV testing history was associated with more sex partners.
- Condom use was associated with fewer sex partners.
- Additional social and behavioral factors may also influence adolescent sexual behavior.
