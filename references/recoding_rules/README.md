# Variable Definitions

## Dataset

**Source:** Youth Risk Behavior Survey (YRBS 2007)

**Population:** Sexually active American adolescents.

---

## Response Variable

### MultipleSexPartners

Definition:
The reported number of sexual partners.

Type:
- Quantitative variable

Role:
- Response (dependent) variable

---

## Predictor Variables

### HIV Education

Definition:
Whether the participant received school-based HIV education.

Type:
- Binary variable

Coding:
- 0 = No
- 1 = Yes

---

### HIV Testing

Definition:
Whether the participant has ever been tested for HIV.

Type:
- Binary variable

Coding:
- 0 = No
- 1 = Yes

---

### Condom Use

Definition:
Whether the participant used a condom during sexual intercourse.

Type:
- Binary variable

Coding:
- 0 = No
- 1 = Yes
