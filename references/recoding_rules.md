# Recoding Rules

## Variable Coding

The predictor variables were recoded into binary variables for regression analysis.

| Variable | 0 | 1 |
|----------|---|---|
| HIV Education | No | Yes |
| HIV Testing | No | Yes |
| Condom Use | No | Yes |

---

## Group Definition

Participants were classified according to HIV-related behaviors.

### HIV Education

- No HIV education
- Received HIV education

### HIV Testing

- Never tested for HIV
- Previously tested for HIV

### Condom Use

- Did not use a condom
- Used a condom

---

## Response Variable

The response variable was:

**MultipleSexPartners**

This variable represents the reported number of sexual partners among sexually active adolescents.

---

## Method Choice

Multiple Linear Regression (OLS) was selected because:

- The response variable is quantitative.
- Multiple predictors were analyzed simultaneously.
- The model estimates the association between HIV-related behaviors and sexual partner numbers.

---

## Assumptions Considered

The following assumptions were considered:

- Linear relationship.
- Independence of observations.
- No severe multicollinearity.
- Homoscedasticity.
- Approximate normality of residuals.

Because the sample size was large (N = 3761), the regression estimates are reasonably robust.
